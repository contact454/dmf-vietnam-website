# Wireframes

Thư mục này chứa các bản phác thảo UI cho website DMF Vietnam.

## Các trang cần wireframe:

1. [ ] Homepage
2. [ ] Khóa học tiếng Đức
3. [ ] Chi tiết khóa học
4. [ ] Du học Đức
5. [ ] Định cư Đức
6. [ ] Tin tức / Blog
7. [ ] Chi tiết bài viết
8. [ ] Về chúng tôi
9. [ ] Liên hệ

## Công cụ đề xuất:
- Figma
- Excalidraw
- Whimsical
- Balsamiq

## Naming convention:
```
[page-name]-wireframe-v[version].png
```

Ví dụ:
- homepage-wireframe-v1.png
- courses-wireframe-v1.png
